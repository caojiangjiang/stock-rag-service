# 批量代码重构技能

## 技能名称
批量代码重构 (Batch Code Refactoring)

## 技能描述
批量修改变量名、函数名、类型名等代码重构任务。能够识别项目中的所有引用位置，并安全地进行重命名，避免遗漏。

## 能力范围

### 1. 重命名类型/结构体
- 修改类型定义名称
- 更新所有引用处的类型名称
- 处理包内和包间的引用

### 2. 重命名函数/方法
- 修改函数定义名称
- 更新所有调用处
- 处理接口实现的方法

### 3. 重命名变量
- 修改局部变量名称
- 更新所有赋值和引用处

### 4. 变量名批量规范
- 驼峰命名 ↔ 下划线命名转换
- 缩写规范化
- 命名风格统一

## 使用限制
- 仅适用于 Go 语言项目
- 需要项目路径
- 仅修改 .go 文件

## 使用示例
```
用户：帮我把 internal/agent.AgentService 重命名为 ChatService
助手：好的，我将执行以下步骤：
1. 查找 AgentService 类型定义
2. 查找所有引用 AgentService 的文件
3. 批量更新所有引用处的名称
4. 编译验证

开始执行...
```
